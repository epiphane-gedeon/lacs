🌐 ARCHITECTURE
════════════════════════════════════════════════════════════════════════════════

Request flow:
──────────────
1. Utilisateur (navigateur 5000)
         ↓ fetch() / Formulaire
2. Flask (test_app.py)
         ├─ Gestion JWT
         ├─ Sessions
         └─ Validation
         ↓ requests
3. API FastAPI (8000)
         ├─ Endpoints
         ├─ Logique métier
         └─ SQLAlchemy ORM
         ↓ SQL queries
4. PostgreSQL (5432)
         └─ Données persistantes
